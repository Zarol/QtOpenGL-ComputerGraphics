# PA7 - Solar System (Ben Nicholes, Denis Morozov, Saharath Kleips)
Interactable solar system created with Qt and OpenGL.

## Extra Credit:
+ Option to go from actual data to a scaled view
+ Uranus has its ring
+ Flythrough camera
+ Skybox
